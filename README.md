# fatitec - Proyecto IHC y Tecnologias moviles

## Descripción del Proyecto
FatiTec es una solución innovadora diseñada para mitigar la fatiga visual mediante técnicas de Interacción Hombre-Computador (IHC), ayudando a estudiantes y trabajadores a gestionar de forma saludable sus hábitos frente a las pantallas.

## Tecnologías Utilizadas
* HTML
* CSS

## Instrucciones de Despliegue
1. Clonar el repositorio localmente usando: `git clone https://github.com/DyCaLaV/fatitec-landing-page.git`
2. Abrir la carpeta del proyecto en Visual Studio Code.
3. Ejecutar el archivo `index.html` en el navegador de su preferencia.
